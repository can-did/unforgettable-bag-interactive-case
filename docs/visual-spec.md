# Visual Specification

## Концепция

Документальная редакционная история, а не SaaS-лендинг. Реальные материалы проекта являются доказательствами и частью повествования.

## Сетка

- максимальная ширина контейнера: 1240 px;
- desktop: 12 колонок;
- tablet: 8 колонок;
- mobile: 4 колонки;
- внешние поля desktop: 48–72 px;
- основная текстовая колонка: не более 680 px.

## Типографика

- H1 desktop: 56–64 px;
- H1 mobile: 38–42 px;
- H2 desktop: 38–44 px;
- H2 mobile: 30–34 px;
- заголовок шага desktop: 24–28 px;
- заголовок шага mobile: 22–24 px;
- основной текст desktop: 18–20 px;
- основной текст mobile: 17–18 px;
- метрики desktop: 64–72 px;
- метрики mobile: 48–56 px.

## Палитра

- фон: `#F5F2EB`;
- основной текст: `#1D1F1C`;
- вторичный текст: `#62665F`;
- разделители: `#D5D2C9`;
- акцент: `#4F6B55`;
- светлый активный фон: `#E4ECE2`;
- ограничения: `#A7663D`;
- фон ограничений: `#F1E6DC`.

## Композиция

- не превращать все элементы в одинаковые rounded cards;
- не использовать тёмный полноширинный финал;
- не использовать pill-navigation как основной визуальный приём;
- не использовать тяжёлые тени;
- не имитировать CRM-интерфейс;
- не использовать фирменные цвета брендов как общий фон;
- метрики на desktop — сетка 2×2;
- основной текст должен читаться без кликов.

## Сквозной блок механики

Task 01 показывает семь шагов открыто и статично.

В последующей задаче:

- desktop: закреплённая визуальная область и последовательные состояния;
- mobile: обычный вертикальный поток без закрепления визуала на весь экран;
- компактный прогресс допустим;
- после первого прохождения возможен возврат к пройденным шагам.

## Движение

- обычный переход: 220–280 ms;
- переход к состоянию отчёта: 400–500 ms;
- без spring, parallax и scroll hijacking;
- обязательная поддержка `prefers-reduced-motion`.

## Доступность

- контраст не ниже WCAG AA;
- видимый focus;
- область нажатия не менее 44×44 px;
- alt-тексты на русском;
- содержательная последовательность DOM;
- отсутствие горизонтальной прокрутки;
- страница понятна без JavaScript.
