# Project brief

Статический интерактивный карьерный кейс «Незабываемая сумка» показывает цепочку: действие покупателя → кассовое событие → обработка → бонус → отчёт → CRM-признак.

Сайт должен быть реализован без фреймворков и сборщика, на семантическом HTML, обычном CSS и минимальном vanilla JavaScript для GitHub Pages.
