# Task 01 — Static foundation

Создать полностью читаемую статическую версию всех утверждённых разделов без фреймворков, сборщика и внешних зависимостей. Основной контент должен быть доступен без JavaScript.

Разрешённые файлы: `index.html`, `styles.css`, `script.js`, `.nojekyll`, `README.md`, `AGENTS.md`, `docs/`, `assets/`.
