# Альтернативные описания изображений

- `01-hero-bag.webp`: Многоразовая сумка проекта с графикой о бонусах.
- `02-qr-tag-closeup.webp`: QR-код с бирки многоразовой сумки крупным планом.
- `03-landing-mechanics.webp`: Фрагмент покупательского лендинга, использованный для объяснения механики.
- `04-bonus-6.webp`: Графический элемент лендинга о начислении 6 бонусов.
- `05-mastercard-bonus.webp`: Графический элемент лендинга о суммарном бонусе при оплате Mastercard.
- `06-store-poster.webp`: Плакат акции «Незабываемая сумка», размещённый в магазине.
- `07-report-redacted.webp`: Обезличенный фрагмент реального транзакционного отчёта по пилоту.
